
# M5 Non-Linear Animation – Fireman Obstacle Course

## Course Information
- **Course:** SPO2002-1 26V CG-art  
- **Assignment:** Mandatory Assignment 5
- **Title:** M5 Non-Linear Animation  

---

## Project Description
This repository contains a **Unity project** focused on creating and implementing **non-linear character animations** using Unity’s **Animator State Machine**. The project features a **fireman obstacle course** animated using a **Fireman Kitten model**.

All **models and animations** are created in **Autodesk Maya**, then imported and used in **Unity**, where they are combined through a state engine and controlled using code.

The objective of the assignment is to demonstrate practical use of animation states, layered animations, transitions, and non-linear animation control in Unity.

---

## Animation Requirements

The following animation sets are implemented in the Unity **State Engine**.


| Primary Animations | Layered Animations |
|-------------------|--------------------|
| Idle              | Carrying Axe       |
| Walk              | Spray Hose Left    |
| Walk Backward     | Spray Hose Right   |
| Run Forward       | Spray Hose Up      |
| Ladder Climb      | Spray Hose Down    |
| Lift              | Spray Hose Straight|
| Axe Swing         | Carry Hose         |
 

These animations are blended and switched using Unity’s Animator Controller to complete different tasks in the fireman obstacle course.
